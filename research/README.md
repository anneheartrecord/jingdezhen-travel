# 瓷都旅游指北研究资料

本目录保存景德镇旅行产品调研资料，供后续路线、景点、住宿和避坑数据迭代使用。

## 文件说明

- `raw/xcrawl-search-results.json`
  - 通过 Xcrawl Search API 批量搜索得到。
  - 覆盖 30 组查询词、299 条原始搜索结果、250 个去重 URL。
  - 只包含公开搜索结果、标题、摘要片段、URL 和查询来源。

- `raw/xcrawl-page-summaries.json`
  - 从重点攻略页提取的摘要和元信息。
  - 覆盖 37 个重点页面，其中 31 个成功获取摘要。
  - 不保存第三方页面全文，只保存摘要、来源 URL、标签和抓取状态。

## 资料分层

- 官方/公共信息：用于景点存在性、假日交通专线、停车与交通公告等高可信事实。
- OTA/攻略平台：用于常见路线、住宿区域、游客偏好、推荐停留时间。
- 社区/游记内容：用于体验型判断，比如打车难、远郊折腾、买瓷器避坑、节假日拥堵。

## 使用原则

- 网站内的时间、费用、交通建议均标记为估算或经验建议。
- 不把小红书、知乎、携程等单一帖子当成绝对事实。
- 涉及门票、开放时间、预约规则时，以官方页面或实时平台为准。
- API Key 不存放在本目录；本机私有 key 存放在 `~/.codex/CODEX.local.md` 或 `.env.local`。
